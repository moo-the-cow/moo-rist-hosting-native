## For all Platforms:
Create a folder `rist-selfhosting` Copy both Zip files into that folder and extract both so it looks like
For Windows
```
rist-selfhosting/
├── librist.zip
├── StatsServer.zip
├── banner.txt
├── stats.json
├── librist
|   ├── tools
|   │   ├── ristreceiver.exe
|   │   └── ristsender.exe
│   └── librist.a
└── StatsServer.exe
```
For linux, linux_arm64, macos
```
rist-selfhosting/
├── librist.zip
├── StatsServer.zip
├── banner.txt
├── stats.json
├── librist
|   ├── tools
|   │   ├── ristreceiver
|   │   └── ristsender
│   └── librist.a
└── StatsServer
```
